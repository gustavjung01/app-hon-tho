UI Ngũ thuật home card assets

Use these to replace the tiny round icons on /nguthuat with larger antique card artwork.

Recommended runtime folder:
apps/web/public/images/nguthuat/home-cards/

Recommended mapping:
Sơn   -> /images/nguthuat/home-cards/ui-nguthuat-card-son-art.png
Y     -> /images/nguthuat/home-cards/ui-nguthuat-card-y-art.png
Mệnh  -> /images/nguthuat/home-cards/ui-nguthuat-card-menh-art.png
Bốc   -> /images/nguthuat/home-cards/ui-nguthuat-card-boc-art.png
Tướng -> /images/nguthuat/home-cards/ui-nguthuat-card-tuong-art.png

Full-card files are for visual reference or for direct image-card designs:
ui-nguthuat-card-*-full.png

Note: Text should remain HTML/CSS in the app, not baked into image assets. Prefer *-art.png for runtime card art.
